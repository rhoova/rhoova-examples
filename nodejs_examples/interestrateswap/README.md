## Run Locally

Clone the project

```bash
  git clone git@github.com:rhoova/rhoova-examples.git
```

Go to the project directory

```bash
  cd rhoova-examples/nodejs_examples/interestrateswap
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
